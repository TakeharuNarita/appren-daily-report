## <li>ソース</li>

*学んだこと（新しい気付き、学び）*

  - 
  - `Server Side Rendering`, `Incremental Static Generation`, `Incremental Static Regeneration`

以下にこれらの特徴を表にまとめてみます。

|          | SSR  | ISG  | ISR  |
|----------|------|------|------|
| レンダリングされる場所  | サーバー | ビルド時 + リクエスト時 | ビルド時 + トリガー時 |
| ユーザーに対する速度  | 高速（初回ロード） | 高速 | 高速 |
| サーバーへの負担  | 高い | 低い | 低い |
| SEO 対策 | 有効 | 有効 | 有効 |
| データの新鮮さ | 高い | 中程度 | 高い |
